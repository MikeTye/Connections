# Connections Wireframe Prototype

This is a runnable React + Vite prototype for the Connections UX wireframe.

## Prerequisites
- Node.js 18+ (recommended: 20+)

## Run locally
```bash
npm install
npm run dev
```

Then open the local URL printed by Vite (typically `http://localhost:5173`).

## Build
```bash
npm run build
npm run preview
```

## Notes
- The prototype emphasizes section-level visibility and viewer-context previews.
- Core components are in `src/components`.
